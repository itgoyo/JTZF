# 🚀 快速开始指南

## 三步开始使用

### 1️⃣ 绑定频道
```
/bind 源频道链接 目标频道链接
```
**示例：**
```
/bind https://t.me/jiaotest0 https://t.me/jiaotest
```

### 2️⃣ 添加关键字过滤（可选）
```
/add 关键字1 关键字2 "多词关键字"
```
**示例：**
```
/add 广告 推广
```

### 3️⃣ 添加文本替换（可选）
```
/replace 原文本 新文本
```
**示例：**
```
/replace www.baidu.com www.google.com
/replace 广告 公告
```

## 重要概念

### 🔄 转发模式
- **黑名单**（默认）：匹配关键字的消息**不转发**
- **白名单**：**只转发**匹配关键字的消息

在 `/settings` 中切换

### 📝 替换规则类型

**1. 普通替换**
```
/replace 广告 公告
```

**2. 链接替换**
```
/replace www.old.com www.new.com
```

**3. 删除文本**
```
/replace 广告
```

**4. 正则替换**
```
/replace https://.*\.com https://new.com
```

## 典型场景

### 场景1：去广告转发
```bash
/bind https://t.me/source https://t.me/target
/add 广告 推广 "点击链接"
```

### 场景2：只转发重要消息
```bash
/bind https://t.me/source https://t.me/target
/settings  # 切换到白名单模式
/add 重要 紧急
```

### 场景3：替换链接和文字
```bash
/bind https://t.me/source https://t.me/target
/replace www.baidu.com www.google.com
/replace 旧品牌 新品牌
```

## 管理多个规则

### 查看所有规则
```
/list_rule
```

### 切换当前操作的规则
```
/switch
```

### 复制规则配置
```
/copy_rule 源规则ID 目标规则ID
```

### 批量操作
```
/add_all 通用关键字        # 添加到所有规则
/replace_all 原文 新文      # 添加到所有规则
```

## ⚠️ 注意事项

1. **权限要求**
   - 机器人需要目标频道管理员权限
   - 用户账号需要加入源频道

2. **当前规则**
   - 使用 `/switch` 查看和切换当前规则
   - 大部分命令都是针对当前规则操作

3. **替换规则**
   - 确保在 `/settings` 中开启"替换模式"
   - 使用 `/list_replace` 确认规则已添加

4. **多词关键字**
   - 使用引号：`/add "这是一个完整的短语"`

## 🆘 遇到问题？

### 消息没转发？
1. 检查规则是否启用：`/settings`
2. 检查关键字匹配：`/list_keyword`
3. 确认转发模式（黑/白名单）

### 替换不生效？
1. 确认替换模式已开启：`/settings`
2. 查看替换规则：`/list_replace`
3. 检查当前规则：`/switch`

### 需要详细帮助？

/help      # 完整命令列表
